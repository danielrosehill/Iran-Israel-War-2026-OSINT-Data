# JSON Exports

Raw wave data in JSON format, directly from the source files.

## Files

### `waves_all.json`
**Combined export** — all 55 waves across TP1–TP4 in a single file with an `export_timestamp` and nested `operations` array. Each operation contains its full `waves` array with all fields.

### `tp1_waves.json`
True Promise 1 — 2 waves, Apr 13–14, 2024. First direct Iranian attack on Israel. ~320 munitions (drones, ballistic missiles, cruise missiles).

### `tp2_waves.json`
True Promise 2 — 2 waves, Oct 1, 2024. Ballistic-missile-only strike. ~200 munitions.

### `tp3_waves.json`
True Promise 3 — 22 waves, Jun 13–24, 2025. "Twelve-Day War". ~1,600–1,800 munitions targeting Israeli cities and military infrastructure.

### `tp4_waves.json`
True Promise 4 — 29 waves, Feb 28–ongoing, 2026. Expanded targeting to US/coalition bases across the Gulf and Mediterranean. Includes Fattah hypersonic missiles, cluster warheads, and Khorramshahr-4.
